(識別コード) (int型の数値1), (int型の数値2), (int型の数値3), (int型の数値4)

-------------------------------------------------

(識別コード)
B：ブロック1つ分の情報
C：ブロックの種類に対応する色

-------------------------------------------------

(int型の数値1)
Bの場合：ブロックの座標x
Cの場合：ブロックの赤色(0～255)

-------------------------------------------------

(int型の数値2)
Bの場合：ブロックの座標y
Cの場合：ブロックの青色(0～255)

-------------------------------------------------

(int型の数値3)
Bの場合：ブロックの座標z
Cの場合：ブロックの緑色(0～255)

-------------------------------------------------

(int型の数値4)
Bの場合：ブロックの種類
Cの場合：ブロックの透明色(0～255)

-------------------------------------------------

ブロックの種類
何も置かない：-1
動かせるブロック：0
動かせないブロック：1
ゴールブロック：2
プレイヤーのスタート位置(描画は無し)：3
ギミック:4～(プログラム上に記述は無いから追加する際は教えて)

-------------------------------------------------

例
B 0, 0, 0, 0
B 0, 2, 1, 1

C 0, 0, 0, 255
C 255, 255, 255, 255
